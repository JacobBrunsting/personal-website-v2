Projects

You can see all of my projects on my Github Profile.

My most recent (and largest) project is a social media website called OpenWindow. It is a location based website that allows you to make text posts, and see and comment on posts from people in your geographic region. It is set up in two layers with a load balancer - one layer to server up the website and direct traffic, and another to manage the storage and backup of the posts. The layers are horizontally scalable, and any server failure is automatically detected and recovered from without any loss of data, in ~30 seconds. It is not currently live, but you can see the code here. It is written in Node.js with MongoDB as the database, and with an Angular 1 frontend (although the front end is a bit neglected).
